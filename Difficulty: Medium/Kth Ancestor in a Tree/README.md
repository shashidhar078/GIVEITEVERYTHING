<h2><a href="https://www.geeksforgeeks.org/problems/kth-ancestor-in-a-tree/1">Kth Ancestor in a Tree</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a <strong>binary tree </strong>with <strong>n</strong> nodes, a target <strong>node</strong> value, and a positive integer <strong>k</strong>, find the <strong>kth ancestor</strong> of the given node.</span></p>
<p><span style="font-size: 18px;">The kth ancestor of a node is the node that lies k levels above it in the tree. If the given node does not have a kth ancestor, return <strong>-1</strong>.</span></p>
<p><strong><span style="font-size: 18px;">Notes</span></strong></p>
<ul>
<li><span style="font-size: 18px;">It is guaranteed that the given node exists in the binary tree.</span></li>
<li><span style="font-size: 18px;">All node values in the tree are <strong>distinct</strong>.</span></li>
</ul>
<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root = [1, 2, 3, 4, 5], k = 2, node = 4<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/700682/Web/Other/blobid0_1745302099.jpg" width="201" height="178"><br></span><span style="font-size: 18px;"><strong>Output:</strong> 1
<strong>Explanation: </strong>The ancestors of node 4 are 2 and 1. Moving 2 levels upward from node 4 reaches node 1, so the output is 1.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root = [1, 2, 3], k = 1, n</span><span style="font-size: 18px;">ode = 3 &nbsp;  <br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927974/Web/Other/blobid1_1778226172.png" height="100"><br></span><span style="font-size: 18px;"><strong>Output: </strong>1
<strong>Explanation: </strong>The parent (1st ancestor) of node 3 is 1. Therefore, the output is 1.<br></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root = [1, 3, 2], k = 3, n</span><span style="font-size: 18px;"><span style="font-size: 18px;">ode = 2  <br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927974/Web/Other/blobid6_1778225798.png" height="100"><br></span><strong style="font-size: 18px;">Output:</strong><span style="font-size: 18px;"> -1<br></span><strong style="font-size: 18px;">Explanation: </strong><span style="font-size: 18px;">Node 2 has only one ancestor, which is 1. Since a 3rd ancestor does not exist, the output is -1.</span></span></pre>
