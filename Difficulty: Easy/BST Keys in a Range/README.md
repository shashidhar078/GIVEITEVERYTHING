<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root = [17, 4, 18, 2, 9, N, N], l = 4, h = 17<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/929462/Web/Other/blobid0_1780918689.webp" width="201" height="130"> <br><strong>Output: </strong>[4, 9, 17]<br><strong>Explanation:</strong> The BST nodes whose values lie in the range [4, 17] are 4, 9, and 17. </pre>

</span></pre>
